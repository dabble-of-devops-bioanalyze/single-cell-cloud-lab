The Single Cell Cloud Lab is created by Dabble of DevOps as a part of the BioAnalyze project.

BioAnalyze is an open source framework for deploying High Performance Compute infrastructure on AWS for Bioinformatics. It uses an analysis first approach to setup sane defaults for an analysis type, such as Single Cell, CellPainting, Clinical Genomics, etc.

CellXGene is an interactive explorer for single-cell transcriptomics data by the Chan Zuckerberg Initiative.
